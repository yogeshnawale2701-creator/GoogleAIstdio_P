

# Yogesh Nawale | DevOps & Cloud Explorer Portfolio

This is the personal portfolio website for Yogesh Vishnu Nawale, a BCA student and DevOps & Cloud learner specializing in Java, PHP, and Cloud Infrastructure.

## Tech Stack
- **Framework:** React 19 with Vite
- **Styling:** Tailwind CSS 4
- **3D Graphics:** Three.js, React Three Fiber, Drei
- **Animations:** Framer Motion
- **AI Integration:** Google Gemini API

## Run Locally

**Prerequisites:** Node.js (v18 or higher recommended)

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in your `.env` or `.env.local` to your Google Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```
3. Run the app:
   ```bash
   npm run dev
   ```

## Scripts
- `npm run dev`: Starts the Vite development server.
- `npm run build`: Builds the app for production.
- `npm run preview`: Locally previews the production build.
